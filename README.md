🌸 Serena — AI-Powered Mental Wellness Platform

Serena is a full-stack AI-powered mental wellness application built using the MERN stack. It helps users improve their emotional well-being through mood tracking, journaling, AI-powered conversations, wellness analytics, and habit building.

✨ Features

* 🔐 Secure JWT Authentication
* 😊 Daily Mood Tracking
* 📖 Personal Journal
* 🤖 AI Wellness Assistant
* 📊 Wellness Dashboard
* 🔥 Streak Tracking
* 💧 Habit Tracking
* 👤 User Profiles

⸻

🛠 Tech Stack

Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* shadcn/ui
* Framer Motion

Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT Authentication
* bcrypt

Clone

Backend

npm install
npm run dev

Frontend

cd frontend
npm install
npm run dev
